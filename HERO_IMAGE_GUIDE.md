# Hero图片使用指南

## 📸 图片要求

首页顶部的Hero区域现在需要一张横向图片。请按照以下要求准备图片：

### 图片规格
- **宽高比**: 推荐 16:9 或 21:9（横向宽图）
- **尺寸**: 建议宽度 1920px 或以上，高度 400-600px
- **格式**: JPG、PNG 或 WebP
- **文件大小**: 建议在 500KB-2MB 之间（可适当压缩）

### 图片内容建议
- 温馨、舒适的场景（如咖啡馆、图书馆、工作室）
- 自然风景（如花园、海滩、山景）
- 抽象艺术或渐变背景
- 与博客主题相关的温馨图片

## 📁 放置位置

将图片命名为 `hero-banner.jpg`，放在前端项目的 `public` 目录下：

```
frontend/
  └── public/
      └── hero-banner.jpg
```

## 🎨 样式说明

如果图片加载失败，系统会自动使用温馨的渐变背景作为备用方案：
- 粉色渐变：`#ffeef8` → `#fff5e6` → `#ffe5e5`

图片上方会有一个半透明的温馨渐变遮罩，确保文字清晰可读。

## 🔧 自定义图片路径

如果想使用不同的图片路径或名称，修改 `frontend/src/views/Home.vue` 中的图片路径：

```vue
<img 
  src="/your-image-name.jpg" 
  alt="欢迎来到我们的博客"
  class="hero-image"
  @error="handleHeroImageError"
/>
```

## 📝 使用外部图片

如果图片托管在外部服务器，可以直接使用URL：

```vue
<img 
  src="https://your-domain.com/images/hero-banner.jpg" 
  alt="欢迎来到我们的博客"
  class="hero-image"
  @error="handleHeroImageError"
/>
```

## ✨ 效果预览

Hero区域现在具有：
- ✅ 全宽横向图片展示
- ✅ 居中的标题和副标题（白色文字，带阴影）
- ✅ 温馨的渐变遮罩
- ✅ 响应式设计（移动端自动调整高度）
- ✅ 优雅的加载失败备用方案

## 🎨 温馨配色方案

整个页面现在使用了温馨的配色：
- **主色调**: 粉色系 (#ffb6c1, #ffb6c1)
- **暖色**: 桃色、橙色渐变
- **背景**: 从白色到温暖的米白色渐变
- **卡片边框**: 淡粉色
- **阴影**: 粉色调的柔和阴影
